# llm-zero-to-one-pytorch

```bash 
pip install uv

uv venv --python=python3.10

source .venv/bin/activate

uv pip install -r requirements.txt

uv run jupyter lab
``` 

```bash
jupyter lab
```

NOTE (On Error): start with full path inside your virtual environment. => .venv/bin/jupyter lab on Linux/macOS or .venv\Scripts\jupyter-lab on Windows.
